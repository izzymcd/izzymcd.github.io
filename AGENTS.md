# Agent Instructions for izzymcd.github.io

A simple static website for "Nails by Izzy" nail business, built with HTML and CSS and hosted on GitHub Pages.

## Project Type

- **Static website** - Plain HTML/CSS, no build process required
- **Hosting**: GitHub Pages (automatic deployment from `main` branch)

## Key Files

- `README.md` - Project documentation
- `index.html` - Main website file (to be created)
- `style.css` - Main stylesheet (to be created)

## Working with This Project

### Deployment
Changes pushed to the `main` branch are automatically deployed to GitHub Pages. The site will be available at `https://izzymcd.github.io`.

### Development
- Use standard HTML5 and CSS3
- No build tools or frameworks required
- Test locally by opening HTML files in a browser

### Best Practices
- Keep HTML and CSS separate for maintainability
- Use semantic HTML for accessibility
- Ensure responsive design for mobile users
- Use CSS custom properties for colors and spacing
- Follow BEM naming convention for CSS classes

## Useful Commands

```bash
# No build commands needed - this is a static site
# Simply push changes to main to deploy
git add .
git commit -m "Your message"
git push origin main
```

## Website Structure

When creating new pages, follow this structure:
```
index.html          - Homepage
services.html       - Services offered
gallery.html        - Nail design gallery
contact.html        - Contact information
css/
  style.css        - Main stylesheet
  reset.css        - CSS reset (optional)
```

## Design Guidelines

### Color Palette
- Soft, feminine colors appropriate for nail business
- Consider: pastels, rose gold, white, soft gray
- Primary accent color: #E8B4B8 (dusty rose)

### Typography
- Clean, elegant fonts
- Headings: serif or display font
- Body: readable sans-serif

### Layout
- Mobile-first responsive design
- Max content width: 1200px
- Standard spacing: 8px base unit
